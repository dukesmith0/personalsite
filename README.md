# Duke Smith - Portfolio

A cinematic, scroll-driven personal portfolio for an aerospace engineer working on satellite instrumentation, built with Vite and vanilla JavaScript: a stylized 3D Earth (Three.js) with glowing vector coastlines, a fresnel atmosphere, a parallax starfield, and a procedural satellite that orbits the globe as the camera flies between sections on scroll (Lenis smooth scroll), graded for a warm "dawn over Earth" look with film grain, fog, and fake-bloom glints; the type is Josefin Sans and Josefin Slab (self-hosted via Fontsource) and all content lives in `src/config.js`, so to run it use `npm install` then `npm run dev` for a hot-reloading dev server, `npm run build` for a production bundle in `dist/`, and `npm run preview` to serve that build, while images drop into `public/images/` and get wired up through the `image` fields in the config.
